# The Heartless
A 2.5D Psychological Horror game built with Python and Ursina Engine.

## Story
You wake up with a hole in your chest. Follow the golden sparks to find your missing heartbeat, only to discover a heartbreaking truth.

## Requirements
- Python 3.10+
- Ursina Engine (`pip install ursina`)

## Controls
- **WASD**: Move
- **Space**: Jump
- **Shift**: Sprint
- **E**: Interact / Read Notes
- **ESC**: Pause
